## JavaScript statement vs expression
  - Expression: A expression generates value and it can be put to any place where need value.for example: 
    1. a
    2. a+b
    3. demo(1)
    4. arr.map()
    5. function test() { }
  - react statement or code just control direction. forexample:
    1. if () {}
    2. for () {} 
    3. .swith(){case: xxx}
